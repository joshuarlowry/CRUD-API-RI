CRM API – Manual CRUD Tests

List all contacts

GET /contacts – should return the current list of contacts.

Create a new contact

POST /contacts – attempts to add a fresh contact with a unique email each time.

Get contact ID 1

GET /contacts/1 – retrieves contact with ID 1. May return 404 if it does not exist.

Delete non-existing contact

DELETE /contacts/9999 – attempts to delete a contact that shouldn't exist and expects 404.
